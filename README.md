# BudgetWise Website

Marketing and landing page for the BudgetWise budgeting app. Features a modern design with dark/light theme toggle and particle animation background.

## Pages

- Homepage with features overview and pricing
- FAQ page
- Privacy Policy
- Terms of Service

## Tech Stack

- HTML, CSS, JavaScript
- Dark/Light theme toggle
- Particle animation (Canvas API)

## Getting Started

Open `index.html` in your browser, or serve with any static file server:

```bash
npx serve .
```
